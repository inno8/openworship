Song Library

Service Schedules

Build setlists ahead of time or on-the-fly. Drag and drop to reorder. Save and reuse for future services.

Multi-Display Output

Project lyrics to any connected screen or projector. Clean, distraction-free output for your congregation.

NDI / OBS Integration

Stream lyrics directly to OBS or any NDI-compatible software. Perfect for livestreaming with transparent overlays.

Keyboard Shortcuts

Navigate with Space, arrows, or press B to black out. Fast controls for smooth service flow.

Customizable Styling

Choose fonts, colors, backgrounds, and text shadows. Make it match your church's visual identity.

Built for Livestreaming

Open Worship outputs lyrics via NDI, so you can overlay them on your livestream without any extra hardware.

  • Lower-third mode (1920×360) for bottom overlays
  • Full-screen mode (1920×1080) for dedicated displays
  • Chroma key support for transparent backgrounds
  • Works with OBS, vMix, Wirecast, and more
  • Zero-latency local network streaming

Get Started in 3 Steps

Up and running in minutes, not hours.

Download & Install

Grab the installer for Windows. Run it and you're ready to go.

Add Your Songs

Type lyrics manually or import from text files. Organize with sections like [Verse] and [Chorus].

Go Live

Build a schedule, connect your projector or OBS, and click GO LIVE. Use Space to advance slides.
