Features

📚

Song Library

Manage lyrics, chords, and media in one place. Import from CCLI, OpenSong, or add your own.

📅

Schedule Builder

Plan services with drag-and-drop. Build setlists and run orders in minutes.

🖥️

Live Presenter

Clean, distraction-free output for projectors and screens. Lyrics, slides, and media in sync.

📡

NDI / OBS Integration

Stream to OBS or any NDI-compatible software. Perfect for online services and recording.
