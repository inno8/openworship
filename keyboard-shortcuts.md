# Keyboard Shortcuts

This page lists the keyboard shortcuts available in Open Worship. They apply mainly in the **Presenter** view when focus is not in a text field (e.g. not in the song search box).

---

## Presentation (Presenter view)

| Shortcut | Action |
|----------|--------|
| **Space** | Next slide. If you’re on the last slide of an item, moves to the first slide of the next schedule item. |
| **→** (Right Arrow) | Same as Space — next slide. |
| **↓** (Down Arrow) | Same as Space — next slide. |
| **←** (Left Arrow) | Previous slide. At the first slide of an item, moves to the previous schedule item. |
| **↑** (Up Arrow) | Same as Left Arrow — previous slide. |
| **B** | Black screen. Sends a black frame to the output. Press again or advance to a lyric slide to show lyrics again. |
| **Enter** | Toggle GO LIVE on/off. If off, turns output on; if on, turns it off. |
| **Escape** | If currently live, stops the presentation output (same as ● STOP). |

---

## Quick reference (one line)

**Navigate:** ← → (or ↑ ↓) • **Next:** Space • **Black:** B • **Live:** Enter • **Stop:** Esc

---

## Where shortcuts work

- **Presenter:** All of the above work when the focus is **not** in an input (e.g. not in the “Search songs…” field). Click in the verse list or on the preview/live area if a shortcut doesn’t respond.
- **Library / Schedule / Settings:** No global presentation shortcuts; use the mouse or normal editing shortcuts in text fields.
